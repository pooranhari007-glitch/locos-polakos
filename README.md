# Locos Polakos

Private summer social club website — yacht parties, rooftop nights, invite-only summer socials.

## Stack

- Next.js 16 (App Router)
- Tailwind CSS 4
- Full-bleed hero video (Pexels yacht party stock footage)

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Design

Mediterranean summer luxury: sand, sea teal, champagne gold. Brand-first hero with looping yacht party video.
